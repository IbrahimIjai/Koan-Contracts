# KoanProtocol

KoanProtocol is a **concentrated liquidity decentralized exchange (DEX)** with gamified features such as a **prediction market** and **lottery game**.  
It includes both **core** and **periphery** smart contracts, deployed on multiple chains.

In-depth documentation is available at: [docs.koanprotocol.xyz](https://docs.koanprotocol.xyz)
